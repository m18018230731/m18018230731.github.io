PostgreSQL 学习笔记 (二) psql工具的使用

  2020-02-03   学习笔记   PostgreSQL  评论 字数统计: 411字 阅读时长: 1分

psql是 PostgreSQL中的一个命令行工具，类似与 Oracle中的 sqlplus，但是与之不同的是 psql支持运行数据库命令或者直接运行 SQL语句并把结果返回。
既然是从头开始学习，当然也把这一部分再巩固下为佳。

进入psql命令行的方法

su - postgres
psql

当显示 postgres=# 时就说明已经进入 psql工具下了。

基础命令

1、查看所有数据库

\l

2、切换数据库

\c testdb

3、查看此数据库下的表

\d

4、查看表的属性

\d [表名]


\d 的用法很多，不仅可以查看表，也可以查看表的索引或是带上通配符查询

\d student_pkey
\d s*

而扩展的 \d 命令则包含了更多功能

\dn         # 列出所有schema
\db         # 显示所有表空间
\du         # 列出所有的数据库用户
\dp [表名]  # 显示表权限分配情况
......

更多的指令在输入 \d 后再补两个 tab键即可查询出来，同期 \t 之类的也类似。
